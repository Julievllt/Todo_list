# Todo_list
A small personal **Todo List** app built with **Python** and **Tkinter**.
Tested on macOS (should work on other platforms).

The UI is displayed on a **Canvas** with a background image and a logo.  
You can add tasks through a popup window and click a task to move it to the **Done** list (with a strikethrough).

## Features
- Add a task via a popup input
- Display tasks on a Tkinter Canvas
- Click a task to mark it as done
- Done tasks are shown with a strikethrough line
- Custom background + logo images
